> The basics to get started on a JavaScript file for SharePoint Online

## Installation

First, install [Yeoman](http://yeoman.io) and generator-spjskickstart using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
npm install -g generator-spjskickstart
```

Then generate your new project:

```bash
yo spjskickstart
```
